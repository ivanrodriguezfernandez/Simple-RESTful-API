### Simple-RESTful-API
Simple RESTful API in Go

### Install Package  
```
go get github.com/gorilla/mux
```
### Build && Run  
```
go build main.go
go run main.go
```
### Docker
```
docker build -t simpleapi .

docker run -it simpleapi

docker run -p 8081:8081 simpleapi
````
### App
http://localhost:8081/users
